# Mina Protocol Integration (o1js)

**Bounty Target:** Mina ($20,000) - ZK App

## Overview
This module contains the design for porting the Obscura verification logic to a Mina zkApp using o1js. This allows users to verify their trading performance on the Mina blockchain, leveraging its constant-size blockchain for lightweight verification.

## Implementation Plan

### 1. Data Structures (Structs)
Define the `Trade` and `PerformanceReport` structs in o1js.

```typescript
import { Field, Struct, UInt64 } from 'o1js';

export class Trade extends Struct({
  timestamp: UInt64,
  entryPrice: UInt64,
  exitPrice: UInt64,
  quantity: UInt64,
  direction: Field, // 0 = Long, 1 = Short
}) {}
```

### 2. ZkProgram
Create a `ZkProgram` that recursively verifies a batch of trades.

```typescript
export const TradeVerifier = ZkProgram({
  name: 'trade-verifier',
  publicInput: Field, // e.g., Hash of the report

  methods: {
    verifyBatch: {
      privateInputs: [TradeBatch],
      method(publicHash: Field, batch: TradeBatch) {
        // Logic to calculate PnL and assert it matches the claim
      }
    }
  }
});
```

### 3. Smart Contract
Deploy a `SmartContract` on Mina that accepts the proof generated by the `ZkProgram` and updates the trader's on-chain reputation.
